Detailed Description

Defaults and settings that affect NetIP.

Some settings affect network communication and should be kept constant for all devices on the network, in particular IP_USE_IPv6 and IP_Buffer_WordCount, but the rest could be (and arguably should be) moved to device-specific application source folders / sketches.
