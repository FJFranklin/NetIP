Detailed Description

A class for storing and manipulating IP addresses. IPv6 addresses are 16 bytes compared to 4 bytes for an IPv4 address. IPv6 protocol headers are 40 bytes vs 20 bytes for IPv4. Support for IPv6 has a significant impact on memory requirements, which is critical for devices such as the Arduino Uno. For this reason, there is support for either IPv4 or IPv6, but not really both. The IP_Address class is defined for one or other, but not both. Perhaps this could/should be rewritten as two distinct classes.
