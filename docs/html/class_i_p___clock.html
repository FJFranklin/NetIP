◆ run()

void IP_Clock::run ( )

Start the infinite loop.

Run the clock. This continues indefinitely, but can be stopped by calling stop(). With each cycle of the infinite loop, tick() is called. Each millisecond, the timers are checked using timer_checks(), and every_millisecond() is called. Once a second, every_second() is called. Subclasses can override tick(), every_millisecond() and every_second().
